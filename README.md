# Tamagotchi-para-microbit
Convierte tu microbit en un tamagotchi

La microbit reaccionara a los inclinamientos a las caidas y al agitarla
El tamagotchi necesita alimentarse salvo que ese durmiendo (presiona a+b)
Para ver su nivel de felicidad presiona B
Para alimentarlo presiona A
Acaricialo para aumentar su felicidad
